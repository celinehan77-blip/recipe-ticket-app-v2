# Recipe Ticket MVP Scope

## 当前 MVP 保留

- 首页粘贴视频链接后进入模拟生成流程。
- Loading 页面继续使用本地模拟任务跳转到菜谱详情页。
- 动态菜谱详情页、动态 Station 页面、风味地图、收藏页和我的页保留。
- 底部 Tab Bar 承担主要导航：首页、收藏、风味地图、我的。
- 数据继续保存在当前浏览器本地，不接后端、不接数据库、不接 AI API。

## 本轮 MVP 简化

1. 首页删除最近生成模块。
2. 首页删除右上角设置 / 筛选按钮。
3. 风味地图分段按钮文案调整为“食材分类 / 烹饪方式”。
4. 风味地图标题改为横向排版。
5. 风味地图删除右侧小票装饰。
6. 我的页删除“查看收藏 / 去风味地图”重复入口。

## 简化原则

- 当前 MVP 保留核心闭环：生成菜谱、查看详情、浏览 Station、收藏菜谱、本地数据汇总。
- 不堆叠重复入口，底部 Tab 已经承担主要导航。
- 搜索、计划、偏好、复杂推荐等功能后期再恢复或增强。
- 第一阶段不接后端、不接数据库、不接 AI API，继续保持本地演示闭环。
