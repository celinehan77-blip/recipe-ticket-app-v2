# AI Knowledge Manager

## 职责

KNOWLEDGE_MANAGER 负责保证代码与项目知识长期一致。

## 必须维护

- `MASTER_PLAN.md`：产品战略，仅在获得战略决策后更新。
- `README.md`：当前能力、运行、环境和部署入口。
- `docs/ROADMAP.md`：Current Phase、版本、状态与 Next Queue。
- `docs/CHANGELOG.md`：实际完成内容和真实验证结果。
- API、数据库、Prompt、部署、测试、设计和 AI 团队文档。

## 工作规则

- 文档描述必须可由代码、配置或验证结果证明。
- 旧文档与代码冲突时，当轮修正或登记明确的文档债务。
- 不记录 Secret、真实 Key、私人数据或未经确认的生产状态。
- 完成 Phase 前执行链接、路径、重复内容和状态一致性检查。
