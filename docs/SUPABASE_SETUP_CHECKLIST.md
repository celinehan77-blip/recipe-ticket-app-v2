# Supabase Setup Checklist

按顺序逐项勾选。

- [ ] 已创建 Supabase 项目。
- [ ] 已打开 Supabase SQL Editor。
- [ ] 已执行 `supabase/schema.sql`。
- [ ] 已执行 `supabase/seed.sql`。
- [ ] Table Editor 能看到 `stations`。
- [ ] Table Editor 能看到 `recipes`。
- [ ] Table Editor 能看到 `ingredients`。
- [ ] Table Editor 能看到 `recipe_steps`。
- [ ] `stations` 里能看到 `chicken`、`pasture`、`seafood`。
- [ ] `recipes` 里能看到 9 道 MVP 测试菜谱。
- [ ] 已复制 Project URL。
- [ ] 已复制 anon public key。
- [ ] 已在项目根目录创建 `.env.local`。
- [ ] `.env.local` 已填写 `NEXT_PUBLIC_SUPABASE_URL`。
- [ ] `.env.local` 已填写 `NEXT_PUBLIC_SUPABASE_ANON_KEY`。
- [ ] 已重启 `npm run dev`。
- [ ] 当前前端仍然正常打开。
- [ ] 当前前端仍然使用 `mockData + localStorage`。
- [ ] 下一步可以开始只读接入 `stations / recipes`。
