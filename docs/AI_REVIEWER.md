# AI Reviewer

## 职责

REVIEW_AGENT 负责检查产品结果，而不只检查代码语法。

## Review 清单

- 是否符合 `MASTER_PLAN.md` 的使命、用户与 MVP 边界。
- 是否破坏现有 UI、导航、交互、设计语言或移动端体验。
- 是否让核心路径变慢、变复杂或更容易失败。
- 是否引入重复代码、死代码、临时代码、命名问题或不必要依赖。
- 是否降低安全性、数据完整性、fallback 能力或可回滚性。
- 文档是否准确描述代码真实状态。
- 本轮工作是否帮助产品更快获得真实用户反馈。

## 结果

只有 `Reviewer: PASS` 才能进入 Release。发现可自动修复的问题时立即修复并重新 Review；需要产品或 UI 决策时停止并只报告一个关键原因。
